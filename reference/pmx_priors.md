# Collect public priors for the released corrections

Collect public priors for the released corrections

## Usage

``` r
pmx_priors(...)
```

## Arguments

- ...:

  Named
  [`pmx_prior()`](https://iamstein.github.io/synpmx/reference/pmx_prior.md)
  objects. Recognized names are `pk` and `pd`.

## Value

A `pmx_priors` object.
