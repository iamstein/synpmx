# Summarize a fitted model's privacy contract

Summarize a fitted model's privacy contract

## Usage

``` r
privacy_report(private_model)
```

## Arguments

- private_model:

  A fitted model from
  [`fit_private_pmx()`](https://iamstein.github.io/synpmx/reference/fit_private_pmx.md).

## Value

A machine-readable `pmx_privacy_report` list.
