# Collect public covariate declarations

Collect public covariate declarations

## Usage

``` r
pmx_covariates(...)
```

## Arguments

- ...:

  Named
  [`pmx_covariate()`](https://iamstein.github.io/synpmx/reference/pmx_covariate.md)
  objects. Each name is a column in the source data and in the generated
  output.

## Value

A `pmx_covariates` object, or `NULL` if nothing is supplied.
